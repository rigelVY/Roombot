## Packages
### roombot_driver
#### Description
Roombaのハードウェア制御用ドライバーパッケージ。
基本的にはRoombaに載っているラズパイ上で動作させる。

なお、Roomba制御用パッケージとしてAutonomyLabさんのcreate_autonomyを使用させて頂いた。
